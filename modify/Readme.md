# RF Switch Tool 變色時間自動化測試系統

本專案是一套基於 Python、Pywinauto 與 Win32 API 驅動的自動化測試系統。系統能自動探索 UI 介面上的 CheckBox 按鈕，依據名稱中的數字進行自然排序，並精準量測按鈕切換時的顏色轉變反應時間（以毫秒 ms 為單位），最後自動將多輪測試結果整合並導出為 Excel 報表。

---

## 系統架構

專案採用三層架構設計，落實職責分離原則，以利後續維護與靈活擴增：

* **核心 (`base_tester.py`)**：負責最底層的 Windows 視窗控制、Win32 API 像素顏色抓取、時間量測計時器以及 Excel 檔案的寫入。
* **UI 探索 (`main.py`)**：透過自動化框架動態掃描軟體畫面上所有「可見（Visible）」且以 `CheckBox` 開頭的控制項，並自動進行數字排序，解除與特定軟體型號的強綁定。
* **測試情境 (`main_test.py`)**：此為測試進入點。負責定義測試的輪次、按鈕點擊的邏輯順序（例如：順序、跳格、隨機），並執行測試循環。

---

## 環境需求與安裝

### 系統環境

* Windows 7 或以上作業系統
* Python 3.8 或以上版本

### 安裝依賴套件

在執行測試前，請開啟終端機（CMD 或 PowerShell）並執行以下指令安裝必備的第三方函式庫：

```bash
pip install pywinauto pywin32 openpyxl numpy
```
